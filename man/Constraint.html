ConstraintR Documentation

Define a set of linearly independent constraints

Description

Define a set of equality, inequality, and/or interval constraints for a parameter of dimension n, using Matrix equation syntax.

Usage

Constraint(
  A = NULL,
  B = NULL,
  C = NULL,
  a = NULL,
  b = NULL,
  c0 = NULL,
  c1 = NULL,
  n = NULL,
  a_default = 0,
  b_default = 0
)

Arguments

A, a

A k \times n matrix A and a n vector a, defining the equality constraint A x = a

B, b

A k \times n matrix B and a n vector b, defining the inequality constraint B x > b

C, c0, c1

A k \times n matrix C and n vectors c0 and c1, defining the interval constraint C x \in (c_0,c_1)

n

An integer greater than 0. The dimension of the parameter. Only needed when A,B, and C are undefined (thus, only for empty constraints)

a_default, b_default

Real numbers. If a or b are left undefined, they default to vectors with n repetitions of a_default/b_default.

Details

The Constraint class is used to define sets of linearly independent equality, inequality, and interval constraints. Constraint objects are created by this function, and by all flavors of helper functions (e.g., EqualityConstraint, LinearConstraint, etc.). Constraints can be combined/intersected with the + operator, and this is the suggested way to build up constraints.

Constraint() acts as the constructor for the Constraint class. Directly using it is the most flexible, but also the least "user-friendly" way to specify constraints (e.g. the exact matrices and vectors need to be figured out beforehand).

Value

An object of class Constraint.
